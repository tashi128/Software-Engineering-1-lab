# Software Engineering Lab

This repository contains the work completed for the **Software Engineering Lab** course. It includes lab exercises, practical implementations, documentation, and project-related materials developed throughout the semester.

## Objectives

The main goals of this lab are to:

- understand core software engineering principles
- apply software development life cycle concepts in practice
- design, implement, test, and maintain software systems
- improve teamwork, documentation, and version control skills

## Contents

This repository may include:

- lab assignments
- mini projects
- source code
- design documents
- UML diagrams
- test cases
- reports and presentations
